# \<chat-bot\>

## Install the Polymer-CLI

```html
<chat-bot></chat-bot>
```

## Install bower
npm i bower

## Run bower install

```
$ bower install
```

|Properties     | Value     | Description                                           |
|-----------    |-----------|-------------------------------------------------------|
|question       |Satring    |This id the text of the node that woild be shown       |
|options        |Map        |This are the options that the bot will show on buttons |
